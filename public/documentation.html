APIs Públicas

GET Público

Chave Pública VAPID

Retorna a chave pública VAPID necessária para configurar notificações push.

Endpoint: /api/vapid-public-key

Resposta: JSON com publicKey

POST Público

Registrar Subscrição

Registra uma nova subscrição de notificação push para um usuário.

Endpoint: /api/subscribe

Parâmetros: userId, subscription, userAgent, etc.

APIs de Autenticação

POST Público

Login Admin

Autentica um administrador e retorna token JWT.

Endpoint: /api/admin/login

Parâmetros: username, password

POST Protegido

Validar Token

Valida um token JWT e retorna informações do admin.

Endpoint: /api/admin/validate

Header: Authorization: Bearer {token}

POST Protegido

Criar Admin

Cria um novo administrador (requer autenticação).

Endpoint: /api/admin/create

Parâmetros: username, password, name

GET Protegido

Listar Admins

Lista todos os administradores cadastrados.

Endpoint: /api/admin/list

Resposta: Lista de admins (sem senhas)

APIs Protegidas (Requerem Autenticação)

GET Protegido

Listar Usuários

Lista todos os usuários inscritos nas notificações.

Endpoint: /api/users

Resposta: Lista de usuários com estatísticas

DELETE Protegido

Remover Usuário

Remove um usuário específico das subscrições.

Endpoint: /api/users/:userId

Parâmetro: userId na URL

POST Protegido

Notificar Usuário

Envia notificação push para um usuário específico.

Endpoint: /api/notify/:userId

Parâmetros: title, body, icon, url, tag

POST Protegido

Notificar Todos

Envia notificação push para todos os usuários ativos.

Endpoint: /api/notify-all

Parâmetros: title, body, icon, url, tag

APIs Super Admin (Máximo Privilégio)

Estas APIs requerem autenticação de Super Administrador e oferecem controle total do sistema.

POST Público

Login Super Admin

Autentica um super administrador e retorna token JWT com privilégios máximos.

Endpoint: /api/super-admin/login

Parâmetros: username, password

Credenciais padrão: superadmin / SuperAdmin@2024!

POST Super Admin

Validar Token Super Admin

Valida um token JWT de super administrador.

Endpoint: /api/super-admin/validate

Header: Authorization: Bearer {token}

POST Super Admin

Criar Admin

Cria um novo administrador com Channel ID seguro.

Endpoint: /api/super-admin/create-admin

Parâmetros: username, password, name

GET Super Admin

Listar Todos os Admins

Lista todos os administradores do sistema com informações completas.

Endpoint: /api/super-admin/admins

Resposta: Lista completa de admins com Channel IDs

PATCH Super Admin

Alterar Status Admin

Ativa ou desativa um administrador específico.

Endpoint: /api/super-admin/admin/:adminId/status

Parâmetros: active (boolean)

GET Super Admin

📊 Visualizar Logs do Sistema

Acessa logs estruturados do sistema para auditoria e monitoramento.

Endpoint: /api/super-admin/logs

Filtros: category, level, limit

Categorias: auth, notification, admin, subscription, tracking, system

Níveis: info, warning, error, security

DELETE Super Admin

🗑️ Limpar Logs Antigos

Remove logs antigos do sistema para otimizar performance.

Endpoint: /api/super-admin/logs/cleanup

Parâmetro: daysToKeep (padrão: 30)
